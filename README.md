该项目是一个打地鼠的游戏。里面共包含4个文件夹music,js,css,img
  文件夹music是用于保存游戏音乐。 
  文件夹js里面有两个JS文件，mouse.js实现老鼠对象功能,main.js实现逻辑功能。
  文件夹css里面是排版设计。
  文件夹img里面存储图片。

游戏规则：打开index.html按照里面红色的游戏说明进行游戏，点击开始进行游戏。直至60S倒计时结束。
